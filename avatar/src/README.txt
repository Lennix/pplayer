EasyWay Game Engine
release 0.3.3.0
http://easyway.sourceforge.net

---- INSTALLATION ----
1) copy all Dlls in the EasyWay folder
2) creates a new project from existing source and select the easyway folder.
3) add the jars to the library project
4) try to run the Main.java Class 

If you have any problems, you can go on Us forum:
http://easyway.sourceforge.net/ewforum
or read the installation tutorial:
http://easyway.sourceforge.net/
